# Online-Parking-Reservation
OOP System Project (2nd year college)

This repository now contains the integrated Parking Reservation system and the
former Information Form applet archive.

Active source code lives in `src/`. The old applet project is kept only for
reference in `legacy/PUP_Information-Form-JApplet` and is not part of the
Eclipse classpath.

Database updates:

- Use `Online Parking Reservation Database/onlineparkingreservation_useraccount.sql`
  for a fresh import with customer occupation/address fields and converted demo
  seed users.
- Use
  `Online Parking Reservation Database/onlineparkingreservation_useraccount_migration_add_profile_fields.sql`
  once on an existing database before running the integrated profile screens.
